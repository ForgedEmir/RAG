# Projet Oracle

## Première étape : établir les fonctionalités.

### Schéma BPMN :

1. L'utilisateur peut poser une question
2. La question est envoyée à l'IA
3. L'IA va chercher les données pour répondre à la question
4. L'IA va généré une réponse
5. La réponse va être envoyée à l'utilisateur
